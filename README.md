# Photoshop 2025 快捷键查询

一个纯 HTML 快捷键查询工具，**无需安装任何依赖**，手机和 PC 浏览器均可直接打开使用。

## 使用方式

直接打开 `index.html` 即可，或将文件传到手机浏览器中。

## 功能

- **131 条快捷键**，覆盖 10 个常用分类
- **实时搜索**：按快捷键名称、按键组合或功能描述筛选
- **分类筛选**：文件、编辑、选择、图层、工具、调整、滤镜、视图、文字、面板
- **深色模式**：跟随系统主题自动切换
- **触控优化**：移动端单手操作适配
- **离线可用**：单文件，不依赖网络

## 快捷键分类

| 分类 | 说明 |
|------|------|
| 文件操作 | 新建、打开、保存、图像大小 |
| 编辑操作 | 撤销、复制、自由变换、填充 |
| 选择操作 | 全选、反选、羽化、快速蒙版 |
| 图层操作 | 复制图层、编组、合并、盖印 |
| 工具快捷键 | V移动、M选框、L套索、B画笔等 |
| 图像调整 | 色阶、曲线、色相/饱和度 |
| 滤镜 | 重复滤镜、液化、Camera Raw |
| 视图控制 | 缩放、适合屏幕、网格、标尺 |
| 文字工具 | 对齐、字号、字距、行距 |
| 面板与窗口 | F5画笔、F6颜色、F7图层等 |
